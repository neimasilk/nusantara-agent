## B. Ruang Lingkup Kerja Individu

| ID ART | Tugas | Target Minimal untuk 1 Orang | Deadline | Status | Catatan |
|---|---|---|---|---|---|
| ART-027 | Seleksi paragraf sumber primer | 60-80 paragraf tervalidasi |  |  |  |
| ART-030 | Kumpulkan putusan MA | 15-20 putusan tervalidasi |  |  |  |
| ART-050 | Susun test case | 40-60 test case + gold reasoning |  |  |  |
| ART-032/033/034 | Kumpulkan korpus domain | 10-15 sumber/domain (pilih 1 domain utama) |  |  |  |

Catatan:
1. Target di tabel adalah target realistis untuk 1 orang.
2. Hasil dari 1 orang bisa digabung dengan pelaksana lain pada tahap integrasi.
